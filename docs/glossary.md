# Glossary

## Context Candidate

A unit of information considered for inclusion in the next model request.

## Artifact

A bulky piece of runtime output stored separately from the prompt payload.

## Surrogate

A compact replacement for a larger artifact.

## Budget

The token allowance for a given turn or agent request.

## Baseline Tokens

The projected token count of the request without optimization.

## Optimized Tokens

The token count after the governor applies its optimization rules.

## Context Slicing

Selecting only the subset of parent context that a subagent actually needs.
